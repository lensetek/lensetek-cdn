# Lensetek CDN

## Ringkasan
Repositori ini menyimpan aset statis, halaman landing, dan materi pembelajaran yang digunakan Lensetek International bersama mitra akademik di Indonesia. Semua konten dapat diakses melalui GitHub Pages (`https://lensetek.github.io/lensetek-cdn/`) untuk kebutuhan promosi program, distribusi materi kuliah, dan referensi visual.

## Struktur Direktori
- **CONMAR/** – Halaman landing "Content Marketing for Emerging Markets" beserta poster promosi. Menyoroti sertifikasi internasional berdurasi 40 jam, kurikulum AI-first, hasil belajar, FAQ, dan tautan pendaftaran untuk mahasiswa Telkom University.
- **MACOMM/** – Halaman landing "Next-Gen Marketing Communication" lengkap dengan poster. Memuat informasi program 40 jam yang berfokus pada komunikasi pemasaran berbasis AI, modul pembelajaran, alur kurikulum, detail sertifikasi, dan proses pendaftaran cohort eksklusif.
- **DIGIMA-U/** – Materi interaktif Modul 3 "Pemasaran Media Sosial" untuk program D3 Digital Marketing. Menyediakan ringkasan tujuan, video pembelajaran, tools AI yang direkomendasikan, studi kasus konten, rencana editorial, tugas praktikum, dan rubrik penilaian.
- **Courses/** – Kumpulan halaman dan data pendukung untuk beberapa mata kuliah bisnis dan pemasaran internasional:
  - `EXIM/4_1` – Materi minggu ke-4 tentang analisis potensi produk ekspor (HTML + JSON rubrik). Menguraikan tujuan, indikator penilaian, daftar tools riset seperti Google Trends dan ITC Trade Map, langkah praktikum, dan keluaran yang diharapkan.
  - `INBUS/7_1` – Landing page minggu ke-7 mata kuliah Bisnis Internasional bertopik Valuta Asing & Pasar Keuangan Internasional. Berisi ringkasan, tujuan pembelajaran, materi inti, latihan, sumber belajar, dan pengingat diskusi kelas.
  - `PPM/4_1` – Materi pertemuan ke-4 Pengantar Pasar Modal mengenai perusahaan efek dan emiten. Menyajikan konsep IPO, peran underwriter dan broker, hak pemegang saham, serta daftar bacaan dan video pendukung.
- **favicon.png**, **logo.png**, dan **AsadelTEK-Logo.png** – Aset branding yang dipakai lintas halaman.

## Cara Menggunakan
1. Buka file HTML secara langsung di peramban atau akses versi publik di GitHub Pages (`https://lensetek.github.io/lensetek-cdn/<path>`).
2. Edit konten melalui HTML/JSON terkait bila ada pembaruan kurikulum, jadwal, atau materi promosi.
3. Gunakan berkas gambar dan ikon yang tersedia untuk memastikan konsistensi identitas visual pada materi tambahan.
